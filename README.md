# WisdomVault - Client

WisdomVault is a full-stack Digital Life Lessons platform where users can create, organize, and share meaningful life lessons with the community. Users can browse public lessons, save favorites, interact through likes and comments, and upgrade to Premium for exclusive content.

```env
BETTER_AUTH_SECRET=

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
```

# 🚀 Run Locally

Clone the project

```bash
git clone https://github.com/mahdihasanprogrammer/Assignment-10-WisdomVault-Client.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build

```bash
npm run build
```

Start

```bash
npm start
```
